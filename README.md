# 🏏 Cricket Highlights System

## 📌 Overview

This project automatically detects important cricket match events like **Four, Six, Wicket, and Crowd Cheer** using video and audio processing.

## 🚀 Features

* Real-time video processing (OpenCV)
* Event detection using motion + audio
* Highlight video generation
* Network streaming support

## 🛠️ Technologies

* C++
* OpenCV
* Boost
* Python

## 📸 Output

* highlights.mp4 (Generated video)
* events.json (Detected events)
